# 《回煞》2D 游戏流程

更新日期：2026-07-11

## 2026-07-12 解谜与恐怖机制

供品不再是落地即拿。厨房需要根据水槽碗痕、灶台灰痕和纸人手势还原饭桌座次；密室需要用封宅笔记和木偶影子拆四方封条；后院需要辨认井绳短结与镇水圆结；阁楼需要从坟位和十年账页中找出无名债主。四件供品齐全后，供桌还要把饭、香、纸钱和火柴对应到留位、引魂、偿债、照真四个位置。

学校记忆改为时间矛盾推理：18:10 仍在留堂、18:20 才有末班车且车程至少二十分钟，因此父亲“18:00 已在家争吵”的说法不成立。医院记忆改为药量推理：十四片每天两片，第五天清晨剩八片，结合病房记录确认第四日晚药盒被父亲取走。谜题完成产生 `caseConclusions`，NarrativeDirector 只回响玩家亲手证明的因果。

新增行为型恐怖：厨房纸人只在玩家背对并移动后推进；学校、医院返回走廊和首次仪式后会发生一次“数脚步”，静止 2.5 秒可避开，继续移动只触发短时黑影与有限理智损失，不重置流程。谜题与对话期间所有行为恐怖暂停，4 秒跨房追逐规则不变。

《回煞》当前是约 20–30 分钟的 2D 民俗恐怖探索游戏。正史为：李明十年前因父亲控制离家并死于车祸；母亲曾自伤获救，之后治疗被父亲阻断，最终投井；父亲死于宅中，留下控制与悔恨化成的黑影。当前雨夜是十周年忌夜的记忆循环，“回煞”指执念沿旧路回返。

## 主流程

入口为 `BootScene -> TitleScene -> IntroScene -> GameScene`。玩家从荒野公路进入老宅，揭开正厅黑布遗像后大门暂时锁死。左上目标面板按状态提示下一步，供品链要求倒头饭、火柴、香和纸钱。

旧书房通向学校记忆。先调查黑板和成绩单，再把 1986、1987、1988 三张试卷按时间排序。药柜小间通向医院记忆。先调查诊室窗口和病房门牌，再按“诊断 -> 处方 -> 未缴费单”拼出治疗被阻断的过程。两段谜题完成后，走廊四张照片会合成残缺全家福。

地下室已改为 18×14 的“地下禁闭室”，没有假岔路；三段暖光串起入口、全家福缺角和返程。缺角与残缺照片合成完整全家福，锁门铁链和墙面刻痕分别补充控制、死亡证据。

后院井边取得血红钥匙后开始短追逐。玩家可以躲藏满 6 秒摆脱黑影，也可以携完整全家福回正厅调查供桌，用照片反制。失败只回当前地图出生点。四样供品齐全后完成仪式，正厅大门重新开放；血红钥匙用于开棺。

## 四结局条件

`StoryState.getExitRoute()` 是唯一出口判定。未完成双谜题时从大门离开为“破茧”；学校和医院谜题完成，但未拼齐全部死亡真相时为“回煞”。完整路线还要求完整全家福、至少两条死亡线索并开棺，随后进入车祸现场。只有变形车门和护栏缺口都调查后，才能选择“归路”或返回记忆空间达成“永远在一起”。

## 维护与验证

源码以 `src/` 为准，`index.html` 由 `node tools/build_standalone_entry.mjs` 重建。地图必须填写非空 `purpose` 和 `rewards`，并通过 `verify_map_purpose.mjs`；进度、谜题、目标、追逐和结局分别由 `verify_progression.mjs`、`verify_puzzles.mjs`、`verify_objectives.mjs`、`verify_chase_contract.mjs` 覆盖。新增持续资源变化必须使用 `delta`，静态体变更后必须同步，持久 DOM 监听必须在场景 shutdown 清理。
